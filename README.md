Cast.us
======================================

**Cast.us** is an Android streaming service integrated with Facebook to enable live group videos from multiple phones.

Feature
-------

- [x] Facebook Login
- [x] Camera Functioning
- [x] Camera Switching
- [x] Adding Friends to the stream
- [x] Inviting friends from friend list
